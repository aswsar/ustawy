# ustawy
Ze względu na fakt, że część parti jest przeciwna jakieś ustawie tylko z powodu zgłoszenia jej przez inną partię proponuję aby rozmawiac tu merytorycznie i unikac wszelakich sympati politycznych aby niezruinowało to możliwości uchwalenia dobrej ustawy przez większość nienarażając się na obawę że sukces zostanie przypisany jakieś innej partji.
